# Human Symptoms-Disease Network (HSDN) Project

## Overview
Replication and extension of "Human symptomsâ€“disease network" (Zhou et al., Nature Communications, 2014).

## Project Goals
- **Replicate** the original HSDN construction and analysis
- **Understand** the methodology through exploratory data analysis
- **Extend** the work with modern approaches and comparisons

## Main Steps

### 1. Replication Phase
- **EDA + Methodology Understanding**
  - Deep dive into paper methodology
  - Data exploration and preprocessing
  
- **Network Reconstruction**
  - Rebuild HSDN using PubMed/MeSH data
  - Implement TF-IDF + cosine similarity approach
  
- **Network Analysis**
  - Reproduce key network properties
  - Replicate main results and figures
  
- **Documentation**
  - Document all methods and findings
  - Ensure reproducibility

### 2. Improvements & Extensions

#### Option A: Comparative Analysis
- **Comparison to Human Disease Network (2007)**
  - Analyze methodological differences
  - Compare network properties and insights

#### Option B: Modern Data Integration
- **Human Phenotype Ontology (HPO) Integration**
  - Validate findings against HPO
  - Enhanced symptom-disease mappings

#### Option C: Applications
- **Drug Discovery Application**
  - Network medicine approach
  - Identify potential therapeutic targets
  
- **OR Graph Neural Networks (GNN)**
  - Apply modern ML to disease-symptom prediction
  - Link prediction and node classification

## Role Division
- **Both team members**: Deep understanding of the paper and methodology
- **Collaborative approach**: Joint responsibility for all phases


## Key Resources
